Question:(1)- What is the difference between Map, Filter, forEach, Find?

ans:- (map) creates a new array by transforming each element of the original array. (filter) creates a new array by selecting only the elements that pass a certain test. (forEach) performs an operation on each element of the array without creating a new array. and (find) returns the first element in the array that satisfies a certain condition.

Question:(2)- Difference between Local Storage, Session Storage, and Cookies

ans:- (Local Storage) and (Session Storage) are both used to store data locally on a user's device. but (Local Storage) persists data across browser sessions while Session Storage is cleared when the browser window or tab is closed. (Cookies) are primarily used for session management, personalization, and tracking, and can be accessed by both the server and the client.

Question:(3)- What is the JavaScript event loop?

ans:- the JavaScript event loop is a mechanism that manages the execution of non-blocking asynchronous operations by maintaining a call stack, a task queue, and a microtask queue, and continuously moving tasks from the queues to the call stack for execution.
